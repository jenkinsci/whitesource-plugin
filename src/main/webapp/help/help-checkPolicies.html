Check Policy Compliance - Check that the introduced open source libraries conform with organization policies.
The plugin will check each new library against the organizational policies.
If any library should be automatically rejected by some policy the build will fail. Otherwise your account will be updated.

Check only new libraries - Check that the newly introduced open source libraries conform with organization policies.
Force check all libraries - Check that all introduced open source libraries conform with organization policies.
Disable - Disable policies check when updating WhiteSource.
